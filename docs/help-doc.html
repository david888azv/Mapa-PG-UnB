Sobre o MAPA-PG

MAPA-PG — Monitoramento e Análise de Produção Acadêmica da Pós-Graduação — é um sistema interativo para acompanhamento e gestão de análise de programas de pós-graduação brasileiros na área de Astronomia / Física.

O sistema permite comparar indicadores de produtividade entre programas com notas CAPES 4, 5, 6 e 7, visualizar tendências ao longo de três quadriênios (2013-2016, 2017-2020, 2021-2024), e gerar relatórios e gráficos exportáveis.

  • Conexão com internet na primeira abertura (para carregar Chart.js via CDN)

Como Usar o MAPA-PG

1. Filtros (Barra Lateral)

  • Notas CAPES: Selecione notas 4, 5, 6 e/ou 7 para filtrar os programas
  • Quadriênio: Escolha um quadriênio específico ou "Todos"
  • Métrica: Selecione a métrica de produtividade (média artigos/docente/ano por categoria)
  • Região: Filtre por região geográfica (CO, NE, N, SE, S)
  • Tipo de produção: Filtre por tipo (Art. Periódico, Resumo, Trab. Congresso, etc.)
  • Estrato IF: Filtre por faixa de fator de impacto (baixo/médio/alto)

2. Resultados

  • Tabela: Ranking dos programas com indicadores completos
  • Gráficos: Barras e evolução temporal por quadriênio
  • Relatório: Texto formatado para cópia e análise

3. Exportação

  • CSV: Exporta todos os dados filtrados em formato CSV (separador: ponto e vírgula)
  • PNG: Exporta todos os gráficos visíveis como imagens PNG com marca d'água
  • Relatório IF: Gera relatório detalhado de fator de impacto em nova janela

4. Destaques

  • A UnB e sempre destacada em rosa nos gráficos e na tabela
  • Clique no cabeçalho de qualquer coluna da tabela para ordenar
  • Os dados de docentes para 2022-2024 utilizam forward-fill do último ano disponível (2021)
Ressalva sobre o número de docentes:
Nos dados brutos disponibilizados pelo Portal de Dados Abertos da CAPES, alguns programas de pós-graduação apresentam o número de docentes igual a zero em determinados anos, especialmente no terceiro quadriênio (Q3, 2021-2024), cujos dados de docentes só foram publicados até 2021. Essa situação pode ocorrer também pontualmente em outros quadriênios para programas específicos.

Para estimar a produção por pesquisador nesses casos, o MAPA-PG adota a seguinte estratégia de forward-fill: quando o número de docentes de um programa é zero em um determinado ano, o sistema utiliza o último registro diferente de zero disponível para aquele programa como substituto. Isso garante que as métricas de produtividade (artigos/docente/ano) possam ser calculadas de forma consistente, mesmo na ausência de dados completos de docentes.

Os valores estimados por este método devem ser interpretados com cautela, pois o número real de pesquisadores ativos pode ter variado no período.
